How to install the sqlite3: http://www.tutorialspoint.com/sqlite/sqlite_installation.htm

1. Go to SQLite download page, and download sqlite-autoconf-*.tar.gz from source code section.  http://www.sqlite.org/download.html

Follow the following steps to install the sqlite:

2. tar xvfz sqlite-autoconf-*.tar.gz
3. cd sqlite-autoconf-*
4. ./configure --prefix=/usr/local
5. make
6. make install
