SRCH2 Documentation

Overview

SRCH2 is a search engine written in C++ ground up. By utilizing advanced indexing structures and algorithms, it delivers superior features with high performance. It runs on both servers as well as mobile devices.

The following figure shows how to use the SRCH2 engine. To use the engine, we provide a configuration file in which we specify parameters for the engine, such as the port numbers, available data sources, and ranking-related values. A data source can be a JSON file or MongoDB, and connectors to other data sources are currently under development.

SRCH2 provides a RESTful API, using which a client can search queries, insert new records, delete existing records, and send other control messages such as saving indexes to the disk.

Features

  • Full-text search
  • Instant search (prefix predicates)
  • Error correction (fuzzy search)
  • Boolean expressions
  • Faceted search
  • Stemming and stop words
  • Phrase search (e.g., "new york")
  • Rich data types (e.g., text, int, float, and date)
  • Attribute-based search predicates
  • Sorting (e.g., "price < 20")
  • Geo search
  • Server-side highlighting
  • Query logging
  • Configurable ranking (including prefix and fuzzy predicates)
  • Real-time updates
  • Multi-threaded
  • Multiple languages
  • Multiple data sources
  • RESTful API
  • Support of control messages (e.g., system status and index serialization)
  • Port-level network security
  • Key-based authorization
  • MongoDB connector, with connectors for MySQL, Sqlite, and other sources coming soon

Supported Platforms

The SRCH2 engine runs on 64-bit Linux (with a kernel version >= 2.6.32), Mac OS (10.5 and above, Intel x86), and Android devices (ARM architecture). It will support iOS in the near future.

The engine supports all Unicode characters encoded using UTF-8.
